:index:`AccountRole Enumeration`
================================

Values
------

======== ==============================================================================
**Name** **Description**
-------- ------------------------------------------------------------------------------
Personal User is required to have a personal account. This matches any Dropbox account.
Work     User is required to have a work (business) account.
